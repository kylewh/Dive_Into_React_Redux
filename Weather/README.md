## 一个天气搜索应用

### Preview:    
![](http://om8hmotom.bkt.clouddn.com/weather.gif)

### Tech Stack:

- react
- redux
- axios
- react-sparklines
- google-map

### 简介
学习过程中的小demo，主要用来练习异步action的发起方法，所以暂时不考虑请求过程中的其他情况，为后期做todo应用打基础。 这里使用是googlemap，weatherAPI使用的也是国外的，所以本地运行时需要翻墙.
